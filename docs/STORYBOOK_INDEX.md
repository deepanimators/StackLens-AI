### Error Detection Pipeline
```
Error in Log ‚Üí LogWatcher Detects ‚Üí Pattern Matching ‚Üí 
Classification ‚Üí ML Scoring ‚Üí Decision ‚Üí Jira Ticket
```

### System Components
1. **Demo POS App** - Generates intentional errors for testing
2. **LogWatcher** - Monitors log files in real-time
3. **Error Detection** - Classifies and analyzes errors
4. **Error Automation** - Makes decisions about errors
5. **Jira Integration** - Creates tickets automatically
6. **Dashboard** - Visualizes all errors and metrics
